# Library Management System
It stores and manages information about books, that include:
- title
- author
- borrowed status (available or borrowed).

## Main features:
It allows the user to:
- add books to the library
- search a book by title
- borrow a book
- return a book
 - print all available books
- print all borrowed books
- count available books
- count books by a specific author
- display all books with their current status.

## Methods:
- borrow() → marks a book as borrowed if available
- returnBack() → returns a borrowed book
- toString() → prints book information
- addBook() → adds a new book to the library
- findByTitle() → searches for a book by title
- printAvailableBooks() → displays all available books
- printBorrowedBooks() → displays borrowed books
- countAvailableBooks() → counts available books
- countBooksByAuthor() → counts books by a given author
- printAllBooksWithStatus() → shows all books with availability status.
